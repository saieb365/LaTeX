\textbf{Symbols for the different number sets displayed using amsfont package:}

\begin{enumerate}

\item The symbol for the set of all natural numbers is: $\mathbb{N}$

\item The symbol for the set of whole numbers is: $\mathbb{W}$

\item The symbol for the set of all real numbers is: $\mathbb{R}$

\item The symbol for the set of all integers is: $\mathbb{Z}$

\item The symbol for the set of all rationals is: $\mathbb{Q}$

\item The symbol for the set of irrational numbers is: $\mathbb{I}$

\item The symbol for complex numbers is: $\mathbb{C}$


\end{enumerate}
